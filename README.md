**HIMS (Hospital Information Management System)** — это современное desktop-приложение для управления базой данных медицинского учреждения. Программа предоставляет удобный интерфейс для работы с пациентами, персоналом, госпитализациями и историей болезни.

Проект разработан с особым упором на **пользовательский опыт (UX)**, производительность и отказоустойчивость.

## ✨ Ключевые особенности (Features)

* 🚀 **Асинхронный UI (Non-blocking):** Все сетевые запросы к API выполняются в фоновых потоках (`QThread` / `ApiWorker`). Интерфейс не зависает во время загрузки данных.
* ⌨️ **Keyboard-First подход:** Полноценная поддержка горячих клавиш. Программой можно управлять без использования мыши (важно для регистратур и врачей).
* 🌗 **Кастомизация:** Встроенный менеджер тем с поддержкой "Тёмного режима" (Dark Mode) для снижения нагрузки на глаза во время ночных дежурств.
* 📊 **Импорт/Экспорт:** Возможность выгрузки любой таблицы в форматы `.csv`, `.xlsx`, `.html` и `.sql`.
* 🛡️ **Защита данных:** Подтверждение критических действий (удаление) паролем, система бэкапов базы данных.
* 🏗️ **Динамические формы:** Формы CRUD (создание, обновление) генерируются автоматически на основе конфигурации таблиц.

## 🛠 Технологический стек

* **Язык программирования:** Python 3.10+
* **GUI Фреймворк:** PyQt6
* **Архитектура:** Клиент-серверная (взаимодействие с Backend через REST API)
* **Асинхронность:** `PyQt6.QtCore.QThread`, `pyqtSignal`
* **Экспорт данных:** встроенные библиотеки `csv`, генерация HTML/SQL.

## ⌨️ Горячие клавиши (Shortcuts)

Для максимальной скорости работы в приложении реализованы следующие шорткаты:

| Комбинация | Действие |
| :--- | :--- |
| `F10` | Активация верхнего меню (навигация стрелочками) |
| `Esc` | Возврат фокуса на основную таблицу |
| `Ctrl + A` | **A**dd (Добавить новую запись) |
| `Ctrl + U` | **U**pdate (Редактировать выбранную запись) |
| `Ctrl + D` | **D**elete (Удалить выбранную запись) |
| `Ctrl + V` | Обновить/перезагрузить текущую таблицу |
| `Ctrl + Shift + E` | Экспортировать данные таблицы |
| `Ctrl + T` | Переключить тему (Светлая / Тёмная) |
| `Ctrl + B` | Создать Backup базы данных |
